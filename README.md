# DecodeLabs Artificial Intelligence Internship

Welcome to the central repository for the **DecodeLabs Artificial Intelligence Internship**. This repository serves as a portfolio of all the projects and tasks completed during the internship program, showcasing progression from foundational logic systems to advanced machine learning pipelines.

---

## 🗂️ Projects Overview

### [Task 1: Rule-Based AI Chatbot](decodelabs_tasks/Task-01-Rule-Based-AI-Chatbot/)
A functional rule-based chatbot that processes user input using conditional logic (`if-else`). It is implemented in two formats:
- **Web Application:** A responsive UI built with HTML, CSS, and Vanilla JavaScript, featuring dark/light mode toggle and interactive elements.
- **Python Script:** A lightweight terminal-based version.

**Key Concepts:** String manipulation, DOM manipulation, conditional decision-making, and responsive frontend UI design.

### [Task 2: Data Classification Using AI](decodelabs_tasks/Task-02-Data-Classification-Using-AI/)
An end-to-end machine learning pipeline built to predict customer order statuses from a synthetic e-commerce dataset. 
- **Models Used:** Decision Tree Classifier & Random Forest Classifier.
- **Techniques Applied:** Exploratory Data Analysis, Feature Engineering, Label Encoding, `StandardScaler`, `GridSearchCV` hyperparameter tuning, and `StratifiedKFold` Cross-Validation.

**Key Concepts:** Data preprocessing, mitigating data leakage, handling high-cardinality data, automated feature selection, and evaluating ML metrics (Precision, Recall, F1-Score).
